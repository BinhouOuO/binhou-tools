# Changelog

All notable changes to this project will be documented in this file. The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-25

### Added
- Initial release of Batch Renamer tool.
- Supports batch renaming for GameObject and Asset selections in Hierarchy and Project views.
- Naming rules: customizable Prefix, Separator, and Index Modes (Numeric / Alphabetic).
- Digit padding option for numeric index mode (e.g. 01, 001).
- Custom sorting modes (Selection Order, Alpha Ascending, Alpha Descending).
- Instant preview for new names before applying.
- Integrated with Unity's Undo system (Ctrl+Z / Cmd+Z support).
